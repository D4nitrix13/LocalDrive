# **En *`tmux`*, puedes buscar texto dentro de un *pane* usando los siguientes atajos:**

1. **Entrar al modo de copia para buscar:**
   - *Presiona `Ctrl-b` seguido de **(shift)** `[` (corchete izquierdo).*

2. **Iniciar la búsqueda:**
   - *Dentro del modo de copia, presiona `/` para buscar hacia adelante o `?` para buscar hacia atrás.*
   - *Escribe el texto que quieres buscar y presiona `Enter`.*

3. **Mover entre las coincidencias:**
   - *Presiona `n` para ir a la siguiente coincidencia.*
   - *Presiona `N` para ir a la coincidencia anterior.*

4. **Salir del modo de copia:**
   - *Presiona `q` para volver al modo normal.*

*Estos pasos te permiten buscar rápidamente texto en cualquier pane abierto dentro de **tmux**. Si buscas una configuración avanzada, puedes personalizar los atajos en tu fichero `.tmux.conf`.*
