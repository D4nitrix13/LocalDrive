# **Calcular cuántos bytes contiene *10 GB* depende de si usas el *sistema decimal (base 10)* o el *sistema binario (base 2)*.**

---

## ***Sistema Decimal (Base 10)***

**En este sistema, se considera que:**

- *`1 GB` = `1,000 MB`*
- *`1 MB` = `1,000 KB`*
- *`1 KB` = `1,000 bytes`*

**Por lo tanto:**

- *`10 GB = 10 × 1,000 MB = 10,000 MB`*
- *`10,000 MB = 10,000 × 1,000 KB = 10,000,000 KB`*
- *`10,000,000 KB = 10,000,000 × 1,000 bytes = 10,000,000,000 bytes`*

**Resultado Decimal:**  

- **`10 GB` = *`10,000,000,000 bytes`* (10 mil millones de bytes)**

---

### ***Sistema Binario (Base 2)***

**En este sistema, se usa *1024* en lugar de *1000*, ya que los datos digitales están basados en potencias de 2.**

- *`1 GB` = `1024 MB`*
- *`1 MB` = `1024 KB`*
- *`1 KB` = `1024 bytes`*

**Por lo tanto:**

- *`10 GB = 10 × 1024 MB = 10,240 MB`*

- *`10,240 MB = 10,240 × 1024 KB = 10,485,760 KB`*

- *`10,485,760 KB = 10,485,760 × 1024 bytes = 10,737,418,240 bytes`*

**Resultado Binario:**  

- **10 GB = *`10,737,418,240 bytes`* (10.73 mil millones de bytes)**

---

### ***¿Cuál usar?***

- *Sistema Decimal:* *Usado en marketing de almacenamiento (como discos duros o unidades flash).*
- *Sistema Binario:* *Usado en sistemas operativos y cálculos técnicos.*
